发表于 2020-12-09
冒泡排序O(n2)

比较相邻的元素，如果前面的比后面的大，就交换他们

选择排序O(n2)

每次从无序区选出最小（大）值放入有序区中

插入排序O(n2)

从后往前，比较有序区，找到位置插入

希尔排序

是插入排序的进化版，又叫缩小增量排序，例如开始分组为

归并排序O(nlogn)

分治法或者二分法

将长度为n的输入序列分为两个长度为n/2的子序列

对子序列分别采用归并排序

将排好的两个子序列归并成一个

快速排序

区一个基数

小于等于这个基数分为一组，大于这个基数分为另一组

每一组再进行快速排序

堆排序O(nlogn)

使用堆这种数据结构，堆排序是一种选择排序

步骤：

  1. 将无序序列构建成一个堆
  2. 将顶元素和末尾元素交换
  3. 出去刚交换的顶元素，剩余重新调整满足堆结构

小顶堆：每个节点都小于等于左右子节点，arr[i] <= arr[2i + 1] && arr[i] <= arr[2i + 2]

大顶堆：每个节点都大于等于左右子节点，arr[i] >= arr[2i + 1] && arr[i] >= arr[2i + 2]

升序采用大顶堆

降序采用小顶堆

计数排序

将输入的数值作为键存在额外的数组空间中，必须有确定范围的整数

步骤：

  1. 算出无序序列范围，如果数值都偏大，可以取余，例如95, 94, 91, 98, 99, 90, 99, 93, 91, 92

    取值范围可以定位0-10

  2. 新建一个数组，初始化为0，例如int arr = new int[10]

  3. 遍历无序序列，和arr下标比较，arr开始计数

  4. 最后按照arr下标取值

桶排序

是计数排序的升级版，利用函数的映射关系，使用链地址法解决冲突的哈希表进行排序

再对每条链进行排序

  • 设置一个定量的数组当作空桶；
  • 遍历输入数据，并且把数据一个一个放到对应的桶里去；
  • 对每个不是空的桶进行排序；
  • 从不是空的桶里把排好序的数据拼接起来。
