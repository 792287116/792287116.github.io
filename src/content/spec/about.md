# 关于这个站点

你好，我是 **Steven**。

这里主要用来记录技术学习、项目复盘、产品想法，以及一些想慢慢打磨清楚的个人笔记。相比一次性的展示页，我更希望它像一座长期维护的写作花园，能持续沉淀思考和经验。

## 这里会写什么

- 项目复盘：做成了什么，卡在哪里，下次怎么做得更好。
- 技术笔记：解决过的 bug、工具使用经验、架构取舍。
- 个人成长：阅读记录、阶段性总结和一些还没完全想透的问题。
